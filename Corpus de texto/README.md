Este apartado contiene el corpus textual original en formato .txt, compuesto por una selección de obras de teatro filipino utilizadas como base para el proceso de anotación y etiquetado lingüístico. Los archivos se presentan en formato plano, sin anotaciones previas, con el objetivo de conservar la estructura original de los textos dramáticos y permitir su posterior procesamiento automático y manual. Este corpus constituye el punto de partida para la creación tanto del gold standard como de las versiones etiquetadas mediante métodos automáticos.
