Este directorio contiene las obras de teatro filipino anotadas automáticamente utilizando el modelo de lenguaje Gemma para el etiquetado y estructuración en formato XML-TEI. El uso de un modelo basado en inteligencia artificial permitió realizar una anotación más flexible y contextualizada en comparación con los métodos basados exclusivamente en reglas, facilitando la identificación de estructuras dramáticas y elementos lingüísticos complejos presentes en los textos teatrales.

Asimismo, se adjunta el código empleado para el etiquetado mediante Gemma, acompañado de un archivo README explicativo donde se describe el flujo completo del procesamiento, la configuración utilizada, las instrucciones de ejecución y las consideraciones metodológicas necesarias para replicar el procedimiento.
